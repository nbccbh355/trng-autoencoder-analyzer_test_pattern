# Example Output – TRNG Test Pattern Analysis

This folder contains example outputs generated using the TRNG test-pattern
preprocessing workflow.

The data shown here is for demonstration and exploratory analysis purposes
only and does not represent production silicon data.

## Command Used

python trng.py --input sample.bin --out out --no-ae

## Included Files

- scores.csv – Summary of SP800-90B style health test results
- notes.md – Observations from pattern-level inspection
